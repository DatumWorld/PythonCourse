
# FastGPT 4.14.5 快速部署

## I 环境
####  windows 11： 不推荐（即使WSL2 Ubuntu也不方便）
#### Ubuntu ：推荐。快速部署推荐docker方式

## II 安装步骤

#### 1 参考官网 https://doc.fastgpt.io/docs/introduction/development/intro
<br> 1 推荐Docker Compose部署后，远程开发。
<br> cd 到 项目根目录/projects/app下面，选择要使用的配置文件，然后运行 sudo docker compose up -d,等待完成（可能要很久，取决于网络链接镜像是否靠谱，靠谱的话一会儿就行）。
<br> 2 等待执行完成（每个项目左边都打对号），访问 http://localhost:3000 即可
 <br>如果不知道端口，执行 sudo docker ps -a，出现类似下图，查找访问端口（这里是默认3000）
